# KPIT Task Workflow - AI Usage

This project was developed with the assistance of **Antigravity**, a high-performance AI coding assistant.

## Contributions of AI

### 1. Rapid UI Development
The AI was used to generate the high-end custom design system, including the glassmorphism effects, gradient palettes, and the responsive layout of the Project Control Center.

### 2. Complex SQL Migrations
The transition from a local-only state to a Supabase-backed architecture was facilitated by AI. It generated the complex RLS policies, PostgreSQL functions, and triggers required for the task workflow.

### 3. Debugging & Error Resolution
AI tools (specifically browser subagents) were used to diagnose and fix critical production bugs, such as:
- **Infinite Recursion in RLS**: Identifying circular dependencies in database policies.
- **State Synchronization**: Fixing mismatches between frontend UI strings and database enum values.

### 4. Boilerplate Generation
Zustand store actions and types were scaffolded by AI to ensure consistent patterns across the application.

## Human Oversight
All code generated by the AI was reviewed for security best practices and logical correctness. The specific business workflow (ToDo -> In Progress -> In Review -> Done) was manually specified and iterated upon to ensure it met the exact user requirements.
